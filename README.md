# Hide Alerts

World of Warcraft Retail addon that blocks achievement alerts and talking head frames.

## Features

- Completely hides achievement alert pop-ups
- Mutes achievement alert sounds
- Blocks talking head frames (NPC dialogue pop-ups)
- Works automatically with no configuration needed
- Eliminates intrusive notifications that can obstruct vision during gameplay

## Compatibility

- **Interface Version**: 11.2.0 (The War Within)
- **Game Version**: World of Warcraft Retail
