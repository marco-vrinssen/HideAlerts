# CleanAlerts

Minimalist World of Warcraft addon that blocks achievement alerts and talking head frames.

## Features

- Completely hides achievement alert pop-ups
- Mutes achievement alert sounds
- Blocks talking head frames (NPC dialogue pop-ups)
- No configuration needed - works automatically

## Installation

1. Download the addon
2. Extract to `World of Warcraft/_retail_/Interface/AddOns/`
3. Restart WoW or reload UI with `/reload`

## Purpose

Eliminates intrusive pop-up notifications that can obstruct vision or distract during gameplay. Perfect for players who prefer uninterrupted gameplay without sudden UI elements appearing on screen.

## Compatibility

- **Interface Version**: 11.2.0 (The War Within)
- **Conflicts**: May conflict with other achievement or dialogue addons
